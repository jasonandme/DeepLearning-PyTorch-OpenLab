# 实验记录

本文件记录当前 18 个 Notebook 的实验目标、设置、观察和可改进点。每个实验都保持小规模，方便本地快速运行和修改。

## 01 Tensor 基础

- 实验目标：理解 Tensor 属性、shape 变化、广播和矩阵乘法。
- 数据设置：手动构造小型 Tensor。
- 模型设置：无模型，关注张量操作。
- 结果观察：shape 是调试 PyTorch 代码的第一线索。
- 可改进点：增加 GPU device 示例。

## 02 Autograd 与计算图

- 实验目标：理解自动求导、梯度累积和最小参数更新。
- 数据设置：标量函数和单参数优化任务。
- 模型设置：手动参数 `w`。
- 结果观察：梯度会推动参数向降低 loss 的方向变化。
- 可改进点：增加 `.detach()` 对计算图影响的实验。

## 03 Module 与训练循环

- 实验目标：使用 `nn.Module` 组织模型并完成标准训练循环。
- 数据设置：一维合成回归数据。
- 模型设置：单层线性回归模块。
- 结果观察：训练循环结构可复用于更复杂模型。
- 可改进点：增加验证集评估。

## 04 Dataset 与 DataLoader

- 实验目标：理解 mini-batch 训练和 DataLoader。
- 数据设置：多特征合成回归数据。
- 模型设置：`nn.Linear(3, 1)`。
- 结果观察：DataLoader 能按 batch 提供数据，并支持 shuffle。
- 可改进点：比较不同 batch size 的训练曲线。

## 05 线性回归

- 实验目标：完成连续值预测闭环。
- 数据设置：`y = 3x + 2 + noise`。
- 模型设置：`nn.Linear(1, 1)`。
- 结果观察：拟合线贴近样本整体趋势，loss 下降但不会为 0。
- 可改进点：增加训练/测试划分和学习率对比。

## 06 Logistic Regression 分类

- 实验目标：理解 logits、类别标签和 `CrossEntropyLoss`。
- 数据设置：二维线性分类数据。
- 模型设置：`nn.Linear(2, 2)`。
- 结果观察：分类任务标签应为 `torch.long`，loss 直接接收 logits。
- 可改进点：加入混淆矩阵。

## 07 MLP 深层网络

- 实验目标：观察 MLP 学习非线性分类边界。
- 数据设置：moon 二维非线性数据。
- 模型设置：两层隐藏层 + ReLU。
- 结果观察：决策边界可弯曲，说明非线性激活发挥作用。
- 可改进点：对比无 ReLU 的模型。

## 08 优化器与学习率

- 实验目标：比较不同学习率和优化器。
- 数据设置：一维合成回归数据。
- 模型设置：线性回归模型。
- 结果观察：学习率直接影响收敛速度和稳定性。
- 可改进点：加入 Momentum SGD。

## 09 正则化

- 实验目标：观察 Dropout 和 weight decay 对泛化的影响。
- 数据设置：小规模 moon 数据。
- 模型设置：较宽 MLP。
- 结果观察：正则化不一定让训练集表现最高，但可能缩小训练/测试差距。
- 可改进点：增加不同 Dropout 概率对比。

## 10 CNN 基础

- 实验目标：观察卷积、池化和特征图 shape。
- 数据设置：手动构造 8x8 灰度图。
- 模型设置：单个卷积层和池化层。
- 结果观察：卷积改变通道数，池化降低空间尺寸。
- 可改进点：对比不同 kernel size 和 padding。

## 11 CNN 图像分类

- 实验目标：训练小型 CNN 完成 digits 分类。
- 数据设置：scikit-learn digits 数据集。
- 模型设置：两层卷积 + 全连接分类头。
- 结果观察：小型 CNN 能快速学习手写数字模式。
- 可改进点：加入数据增强或更多卷积通道。

## 12 CNN 特征可视化

- 实验目标：观察中间特征图和错误样本。
- 数据设置：digits 数据集。
- 模型设置：浅层 CNN。
- 结果观察：错误样本分析比只看 accuracy 更有助于理解模型行为。
- 可改进点：统计类别混淆情况。

## 13 RNN 基础

- 实验目标：理解 RNN 输入输出和隐状态。
- 数据设置：随机序列和简单正弦序列窗口。
- 模型设置：`nn.RNN`。
- 结果观察：RNN 输出包含每个时间步的信息。
- 可改进点：将 RNN 用于完整预测任务。

## 14 LSTM 时间序列预测

- 实验目标：使用历史窗口预测下一个序列值。
- 数据设置：带噪声正弦序列。
- 模型设置：单层 LSTM + 线性输出层。
- 结果观察：预测曲线能跟随主要趋势，但不会完全贴合噪声。
- 可改进点：尝试 GRU 和多步预测。

## 15 从零实现 Attention

- 实验目标：理解 Q/K/V 和 scaled dot-product attention。
- 数据设置：手动构造短序列表示。
- 模型设置：手写 attention 计算。
- 结果观察：每一行 attention weight 表示一个 query 对所有 key 的关注分布。
- 可改进点：加入 causal mask。

## 16 Multi-Head Attention

- 实验目标：使用 `nn.MultiheadAttention` 并观察 mask 行为。
- 数据设置：随机 token 表示。
- 模型设置：多头注意力模块。
- 结果观察：mask 可让模型忽略 padding 位置。
- 可改进点：可视化不同 head 的权重。

## 17 Transformer Encoder

- 实验目标：搭建最小 Encoder Block。
- 数据设置：随机序列表示。
- 模型设置：MHA + 残差连接 + LayerNorm + FFN。
- 结果观察：Block 输入输出 shape 保持一致，因此可以堆叠。
- 可改进点：加入分类头完成序列分类。

## 18 训练工程

- 实验目标：记录日志、保存 checkpoint、加载模型状态。
- 数据设置：合成分类数据。
- 模型设置：小型 MLP。
- 结果观察：日志和 checkpoint 能让实验更容易复盘和恢复。
- 可改进点：保存最佳验证集 checkpoint，并导出 CSV 日志。
