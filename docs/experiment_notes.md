# 实验记录

这份文档记录项目中主要实验的目标、设置、观察和可改进方向。实验规模保持较小，重点是让读者能运行、能观察、能复盘。

## 1. Tensor 与 Autograd 基础

**实验目标**  
理解 Tensor 运算和自动求导之间的关系。

**数据设置**  
使用手动构造的小型 Tensor，不依赖外部数据。

**模型设置**  
没有完整模型，只使用一个可学习标量参数观察梯度更新。

**训练过程**  
构造简单预测值和目标值，计算平方误差，调用 `backward()` 得到梯度，再手动更新参数。

**结果观察**  
参数会沿着降低 loss 的方向变化，loss 逐步下降。梯度清零前后 `.grad` 的变化也能直观看到。

**结论**  
Autograd 可以自动完成梯度计算，但梯度累积和计算图管理仍需要明确理解。

**可改进点**

- 增加更多广播和矩阵乘法示例。
- 增加 `.detach()`、`no_grad()` 和计算图断开的对比实验。

## 2. 线性回归实验

**实验目标**  
用简单连续值预测任务跑通第一个完整训练闭环。

**数据设置**  
使用合成数据 `y = 3x + 2 + noise`。噪声用于模拟真实数据中的随机扰动。

**模型设置**  
手动定义参数 `w` 和 `b`，预测形式为 `y_hat = w * x + b`。

**训练过程**  
使用 MSELoss 衡量预测误差，通过梯度下降更新 `w` 和 `b`，记录每个 epoch 的 loss。

**结果观察**  
loss 曲线整体下降，拟合直线贴近散点整体趋势。学到的参数通常接近数据生成时的主要线性关系。

**结论**  
线性回归适合理解训练闭环。即使模型简单，也能覆盖前向传播、loss、反向传播和参数更新。

**可改进点**

- 对比手写参数和 `nn.Linear` 实现。
- 增加不同学习率下的 loss 曲线对比。
- 增加训练集和测试集划分。

## 3. MLP 分类实验

**实验目标**  
观察 MLP 如何通过激活函数拟合非线性分类边界。

**数据设置**  
使用二维 moon 合成数据，并划分训练集和测试集。输入特征经过标准化处理。

**模型设置**  
使用两层隐藏层和 ReLU 激活函数，最后输出两个类别的 logits。

**训练过程**  
使用 `CrossEntropyLoss` 训练分类模型，优化器使用 Adam，记录训练 loss，并在测试集上计算 accuracy。

**结果观察**  
loss 逐步下降，测试准确率通常较高。决策边界呈现弯曲形状，能更好适应非线性数据分布。

**结论**  
MLP 的关键是激活函数带来的非线性表达能力。分类任务中 logits、label dtype 和 loss 函数必须正确匹配。

**可改进点**

- 对比没有激活函数的模型表现。
- 增加隐藏层宽度和深度的对比。
- 增加过拟合与正则化观察。

## 4. CNN 图像分类实验

**实验目标**  
使用小型 CNN 观察卷积层如何处理图像局部结构。

**数据设置**  
使用 scikit-learn 内置 digits 数据集。图像大小为 `8 x 8`，输入 shape 转换为 `[N, 1, 8, 8]`。

**模型设置**  
模型包含两个卷积层、ReLU、MaxPool 和全连接分类头。

**训练过程**  
使用 mini-batch 训练，loss 函数为 `CrossEntropyLoss`，优化器为 Adam。训练后在测试集上计算 accuracy。

**结果观察**  
loss 曲线明显下降，测试准确率通常较高。错误样本多出现在形状相似的数字之间。

**结论**  
CNN 能利用局部连接和参数共享处理图像结构。写 CNN 时必须认真跟踪每层输出 shape。

**可改进点**

- 增加特征图可视化。
- 对比 MLP 和 CNN 在同一数据集上的表现。
- 增加数据增强实验。

## 5. RNN/LSTM 序列建模实验

**实验目标**  
用 LSTM 根据历史窗口预测序列的下一个值。

**数据设置**  
使用带噪声的正弦序列，通过滑动窗口构造输入和目标。输入 shape 为 `[batch, window_size, 1]`。

**模型设置**  
使用单层 LSTM 表示序列信息，再用线性层输出一个连续预测值。

**训练过程**  
使用 MSELoss 训练，记录训练 loss，并在测试窗口上生成预测曲线。

**结果观察**  
预测曲线能跟随目标序列的大致趋势，但在噪声和局部波动处存在误差。

**结论**  
LSTM 能利用历史窗口进行序列预测。窗口大小、隐藏维度和学习率会明显影响结果。

**可改进点**

- 对比 RNN、GRU 和 LSTM。
- 尝试不同窗口长度。
- 增加多步预测实验。

## 6. Attention 权重可视化实验

**实验目标**  
理解 scaled dot-product attention 的计算流程和权重矩阵含义。

**数据设置**  
使用手动构造的小型序列 Tensor，便于观察每个位置之间的权重。

**模型设置**  
先手写单头自注意力，再使用 `nn.MultiheadAttention` 构造小型 Transformer Block。

**训练过程**  
本实验不进行训练，重点是前向计算和 shape 观察。

**结果观察**  
注意力权重矩阵的每一行都可以看作一个位置对其他位置的关注分布。Transformer Block 的输入输出 shape 保持一致。

**结论**  
Attention 可以看作基于相关性的加权汇总机制。理解 Q/K/V 和权重矩阵，是学习 Transformer 的基础。

**可改进点**

- 增加多头注意力中不同 head 的权重对比。
- 增加位置编码示例。
- 在小型序列任务上训练简化 Transformer。
