# 深度学习概念问答

这份文档用于快速复习深度学习和 PyTorch 基础概念。回答尽量保持简洁，适合在完成 Notebook 后回顾。

## PyTorch 基础

**Q1：Tensor 是什么？**  
Tensor 是 PyTorch 中表示多维数组的对象。输入数据、模型参数、中间特征和梯度都可以用 Tensor 表示。

**Q2：Tensor 和 NumPy array 有什么区别？**  
二者都能表示多维数组。Tensor 支持 GPU 计算和自动求导，更适合深度学习训练。

**Q3：为什么 shape 很重要？**  
模型层对输入维度有明确要求。shape 不匹配会导致运行错误，或者触发意外广播造成结果不正确。

**Q4：`nn.Module` 的作用是什么？**  
`nn.Module` 用来组织模型结构和参数。自定义模型通常继承它，并在 `forward()` 中定义前向计算。

**Q5：`state_dict()` 保存的是什么？**  
它保存模型或优化器的参数状态，常用于模型保存、加载和继续训练。

## Dataset/DataLoader

**Q1：为什么需要 DataLoader？**  
DataLoader 可以按 batch 提供数据，并支持 shuffle、多进程读取等功能，让训练流程更接近真实项目。

**Q2：batch size 会影响训练吗？**  
会。较小 batch 更新更频繁但波动更大，较大 batch 更稳定但可能需要更多内存。

**Q3：训练集为什么通常要 shuffle？**  
shuffle 可以减少样本顺序对参数更新的影响，让 mini-batch 更接近整体数据分布。

**Q4：测试集需要 shuffle 吗？**  
通常不需要。测试阶段不更新参数，保持固定顺序更方便复现和分析。

## 自动求导

**Q1：`requires_grad=True` 表示什么？**  
表示 PyTorch 需要跟踪这个 Tensor 参与的可求导运算，并在反向传播时计算梯度。

**Q2：`backward()` 做了什么？**  
它从当前标量 loss 出发，沿计算图反向计算各参数的梯度。

**Q3：梯度保存在哪里？**  
梯度通常保存在对应参数的 `.grad` 属性中。

**Q4：为什么梯度会累积？**  
PyTorch 默认累积梯度，方便支持梯度累积等训练方式。普通训练循环中需要手动清空。

**Q5：什么时候使用 `torch.no_grad()`？**  
评估、推理或手动更新参数时不需要构建计算图，可以使用 `torch.no_grad()` 减少内存占用。

## 梯度下降

**Q1：梯度下降的核心思想是什么？**  
根据 loss 对参数的梯度，沿着让 loss 减小的方向更新参数。

**Q2：学习率控制什么？**  
学习率控制每次参数更新的步长。太大可能震荡，太小可能收敛很慢。

**Q3：为什么要调用 `optimizer.zero_grad()`？**  
因为梯度默认累积。如果不清零，当前 batch 的梯度会和之前的梯度叠加。

**Q4：SGD 和 Adam 有什么区别？**  
SGD 更新规则简单，Adam 会结合一阶和二阶动量信息，通常更容易在初始实验中得到稳定结果。

**Q5：loss 不下降一定是模型不行吗？**  
不一定。也可能是学习率、数据预处理、标签格式、loss 使用方式或训练循环有问题。

## 优化器与正则化

**Q1：SGD 和 Adam 应该怎么选？**  
入门实验中 Adam 往往更容易稳定收敛；SGD 更简单，也常用于理解梯度更新本身。

**Q2：学习率如何初步设置？**  
可以从 `1e-3` 或 `1e-2` 试起，再根据 loss 曲线调整。震荡明显就降低，下降太慢可适当提高。

**Q3：什么是过拟合？**  
模型在训练集表现很好，但在测试集表现明显较差，说明它可能记住了训练数据细节。

**Q4：Dropout 做什么？**  
训练时随机丢弃部分神经元输出，降低模型对局部特征组合的依赖。

**Q5：weight decay 做什么？**  
它通过惩罚较大的参数值来限制模型复杂度，是常见正则化方法。

## 线性回归

**Q1：线性回归学习什么？**  
学习输入和连续目标之间的线性关系，常见形式是 `y = wx + b`。

**Q2：为什么线性回归常用 MSELoss？**  
MSELoss 衡量预测值和真实值的平方误差，适合连续值预测。

**Q3：线性回归实验的重点是什么？**  
重点是理解完整训练闭环，而不是模型本身有多复杂。

**Q4：为什么参数不一定等于数据生成时的真实值？**  
如果数据中有噪声，模型学到的是在当前样本上的较优近似，不一定完全等于真实参数。

## MLP

**Q1：MLP 为什么需要激活函数？**  
没有激活函数时，多层线性层仍等价于一个线性变换。激活函数让模型具备非线性表达能力。

**Q2：ReLU 的作用是什么？**  
ReLU 将负数截断为 0，形式简单，计算高效，是常见的非线性激活函数。

**Q3：logits 是什么？**  
logits 是分类模型输出的原始分数，通常直接传给 `CrossEntropyLoss`。

**Q4：为什么 `CrossEntropyLoss` 前不要手动 softmax？**  
因为 `CrossEntropyLoss` 内部已经包含相关计算，提前 softmax 可能影响数值稳定性。

**Q5：隐藏层越多越好吗？**  
不一定。更深的模型表达能力更强，但也更难训练，且更容易过拟合。

## CNN

**Q1：CNN 适合图像任务的原因是什么？**  
CNN 能利用图像局部结构，并通过参数共享减少参数量。

**Q2：卷积层输出的通道数表示什么？**  
输出通道数可以理解为卷积层学习到的特征图数量。

**Q3：池化层有什么作用？**  
池化层通常用于降低空间尺寸、减少计算量，并提升一定的位置鲁棒性。

**Q4：图像输入为什么通常是 `[N, C, H, W]`？**  
`N` 是 batch 大小，`C` 是通道数，`H` 和 `W` 是图像高度和宽度。

**Q5：CNN 中最常见的 shape 错误是什么？**  
忘记 channel 维度，或者 flatten 后的特征数与 `nn.Linear` 的 `in_features` 不一致。

## RNN/LSTM

**Q1：RNN 适合什么数据？**  
RNN 适合有顺序依赖的数据，例如时间序列、文本和语音。

**Q2：LSTM 相比普通 RNN 的改进是什么？**  
LSTM 使用门控机制控制信息保留和遗忘，能更稳定地处理较长依赖。

**Q3：`batch_first=True` 有什么作用？**  
它让输入 shape 使用 `[batch, seq_len, features]`，更符合多数数据处理习惯。

**Q4：滑动窗口在序列预测中做什么？**  
它把连续序列切成输入窗口和预测目标，让序列预测变成监督学习问题。

**Q5：为什么序列预测不一定完全贴合目标曲线？**  
噪声、窗口长度、模型容量和训练设置都会影响预测效果。趋势正确通常比逐点完全重合更重要。

## Attention/Transformer

**Q1：Attention 的核心思想是什么？**  
根据相关性为不同位置分配权重，再对信息进行加权汇总。

**Q2：Q、K、V 分别表示什么？**  
Query 表示查询，Key 用于匹配查询，Value 是被加权汇总的信息。

**Q3：为什么要除以 `sqrt(d_k)`？**  
当向量维度较大时，点积值可能过大，缩放可以让 softmax 更稳定。

**Q4：Multi-head Attention 的直观理解是什么？**  
多个 attention head 可以从不同子空间观察序列关系，再把结果合并。

**Q5：Transformer Block 通常包含哪些部分？**  
通常包含多头注意力、残差连接、LayerNorm 和前馈网络。

**Q6：Attention 权重能完全解释模型吗？**  
不能。权重可以提供观察角度，但模型输出还受到投影层、前馈网络和训练目标等因素影响。

## 训练工程

**Q1：为什么要记录训练日志？**  
日志能帮助复盘 loss、accuracy 和训练设置，避免只凭最终结果判断模型。

**Q2：checkpoint 保存什么？**  
通常保存模型参数、优化器状态、epoch、训练日志和必要配置。

**Q3：为什么要固定随机种子？**  
固定随机种子可以减少随机初始化、数据划分等因素带来的结果波动，方便复现实验。

**Q4：什么时候保存最佳模型？**  
通常根据验证集指标保存，而不是只保存最后一个 epoch 的模型。
